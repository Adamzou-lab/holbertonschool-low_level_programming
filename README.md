# Holberton Low Level Programming

Apprentissage du langage C à Holberton School, des bases jusqu'aux structures de données. Chaque dossier est un projet indépendant, avec ses fichiers source et son fichier d'en-tête.

## Projets

| Dossier | Contenu |
|---|---|
| `hello_world` | Premiers programmes, compilation et affichage |
| `variables_if_else_while` | Variables, conditions et boucles |
| `functions_nested_loops` | Fonctions et boucles imbriquées |
| `more_functions_nested_loops` | Fonctions supplémentaires, dessins ASCII |
| `pointers_arrays_strings` | Pointeurs, tableaux et chaînes de caractères |
| `recursion` | Fonctions récursives |
| `argc_argv` | Arguments de la ligne de commande |
| `malloc_free` | Allocation dynamique de mémoire |
| `more_malloc_free` | Allocation dynamique avancée |
| `function_pointers` | Pointeurs de fonctions |
| `variadic_functions` | Fonctions à nombre variable d'arguments |
| `structures_typedef` | Structures et typedef |
| `singly_linked_lists` | Listes chaînées simples |
| `doubly_linked_lists` | Listes doublement chaînées |
| `hash_tables` | Tables de hachage |
| `file_io` | Lecture et écriture de fichiers |
| `intro_debugging` | Introduction au débogage |
| `dynamic_analysis` | Analyse dynamique de code |
| `benchmarking` | Mesure de performance |
| `secure_data_handling` | Manipulation sécurisée de données |
| `simple-calculator` | Petite calculatrice en C |
| `ai_memory_visualizer_valgrind_tracer` | Visualisation mémoire et trace Valgrind |

## Compilation

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o programme
```

## Prérequis

- Ubuntu 20.04 LTS
- GCC, standard gnu89
